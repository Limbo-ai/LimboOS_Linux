# LimboOS_Linux

LimboOS is a fully open-source operating system based on Arch Linux. It is my personal passion project, born from sleepless nights, gallons of Monster Energy, and the sheer will to build a system exactly the way I want it.

## >_ The Origin
*   **Built on:** Arch Linux.
*   **Fuel source:** Endless cans of Monster Energy and a total disregard for sleep schedules.
*   **Status:** A pure labor of love, developed during countless late-night sessions.

## >_ Technical Core
LimboOS features a highly customized environment optimized for stability and my personal workflow:

*   **Kernel:** Linux 6.19.14-arch1.
*   **DE/WM:** KDE Plasma 6.6.4 with KWin (Wayland) and a bespoke 'blackglass' WM theme.
*   **Visuals:** Custom Galaxy branding, Papirus-Dark-Grey icons, and a Skyrim-themed cursor.
*   **Foundation:** Currently running on an AMD Ryzen 7 5700G.

## >_ Why?
Why? Because I could. Because I wanted my system to look and function exactly how I envisioned it—from the very first line of the GRUB bootloader to the final terminal command.
